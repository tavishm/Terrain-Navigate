# Synthetic Examples & Tutorials

This directory contains tutorials for learning the `terrain_navigate` library using synthetic data.

## Notebooks

*   **`tutorial.ipynb`**: The main tutorial.
    *   Generates synthetic terrain (hills, obstacles).
    *   Visualizes terrain in 3D.
    *   **Compares Cost Functions**: Shows how `PowerCost` avoids slopes while `EuclideanCost` takes the shortest path.
    *   Demonstrates customization options.

## Usage

Just open `tutorial.ipynb` in Jupyter! No data download required.
